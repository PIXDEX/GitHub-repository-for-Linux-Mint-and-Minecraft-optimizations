### Basic Cheat code for Linux mint

* **cd:** Muévete por los directorios

* **ls:** Muestra una lista de los elementos que hay en tu carpeta actual

* **cp** copia cosas al directorio actual (cp directory/object.ob) *tiene uso especial y es que si quieres renombrarlo al final pon por lo que quieres renombrar*

* **mv:** mueve archivos muy util para una transferencia rapida

* **apt install (Repository):** es una manera de descargar archivos de programas Software
* 
* **apt-get install (Repository):** es otra manera dé descargar cosas pero de diferente manera y repositorios ***No recomendado método algo antiguo!***
* 
* **htop:** es una manera de ver componentes y el consumo de ellos. **Requiere Instalar htop desde "apt install htop"**
